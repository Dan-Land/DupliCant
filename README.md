# DupliCant

A Python command-line utility for identifying duplicate files.

## Status

This project is currently under development.

## Planned Features

- Scan a user-selected directory
- Identify likely duplicates by file size
- Confirm duplicates using cryptographic hashes
- Display groups of matching files
- Optionally export the results
- Avoid deleting files without explicit user confirmation

## Motivation

DupliCant is being developed as a practical Python project focused on file-system operations.
